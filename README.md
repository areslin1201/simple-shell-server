# ⚡ ShellHub — Shell Script 管理與執行平台

一個基於 **React + Express** 的 Web 平台，可透過瀏覽器管理與即時執行 Shell Script，支援分類管理、參數輸入、SSE 串流即時輸出。

---

## 🚀 快速開始

### 一鍵安裝所需環境

```bash
./first-time-install.sh
```

### 啟動

```bash
docker-compose up -d
```

開啟前端連結：[http://localhost:5173](http://localhost:5173)

### 暫停

```bash
docker-compose stop
```
